Негізгі ұғымдар

Ньютон заңдары

Инерция заңы

Егер денеге басқа денелер әсер етпесе, онда дене тыныштық күйінде немесе түзусызықты бірқалыпты қозғалыста болады.

∑F = 0 ⇒ a = 0
Динамиканың негізгі заңы

Дененің үдеуі оған әсер ететін күшке тура пропорционал және дененің массасына кері пропорционал.

F = m·a
Әсер және қарсы әсер заңы

Екі дене бір-біріне шамалары бірдей, бағыттары қарама-қарсы күштермен әсер етеді.

F₁₂ = -F₂₁

Күштер түрлері

Ауырлық күші

Жердің тартылыс күші

F = m·g

g = 9.8 м/с² - еркін түсу үдеуі

Серпімділік күші

Деформацияланған дененің қалпына келу күші

F = -k·x

k - қаттылық коэффициенті, x - ұзару

Үйкеліс күші

Қозғалысқа қарсы бағытталған күш

F = μ·N

μ - үйкеліс коэффициенті, N - қалыпты қысым күші

Күштер диаграммасы

Көлбеу жазықтықтағы дене

Ауырлық күші (mg)

Қалыпты қысым күші (N)

Үйкеліс күші (Fүйк)

Күштерді талдау әдісі:
  1. Денеге әсер ететін барлық күштерді анықтау
  2. Күштерді координаталық осьтерге проекциялау
  3. Ньютонның екінші заңын әр оське қолдану
  4. Теңдеулер жүйесін шешу
Көлбеу жазықтық үшін:

mg·sinα - Fүйк = m·a

N = mg·cosα

Импульс пен энергия

Импульс

Дененің қозғалыс мөлшері

p = m·v

Импульстің сақталу заңы:

∑pалдында = ∑pкейін

Қолданысы:

  • Соқтығысуларды зерттеу
  • Реактивті қозғалыс
  • Ракеталардың ұшуы
Энергия

Дененің жұмыс істеу қабілеті

Кинетикалық энергия:

Eк = (m·v²)/2

Потенциалдық энергия:

Eп = m·g·h

Энергияның сақталу заңы:

Eалдында = Eкейін

Практикалық есептер

Есеп 1: Көлбеу жазықтық

Массасы 2 кг дене 30° бұрыш жасайтын көлбеу жазықтықтан сырғанап түсіп жатыр. Үйкеліс коэффициенті 0.1. Дененің үдеуін табыңыз.

Шешімі:

1. Күштерді проекциялау:

mg·sinα - Fүйк = m·a

N = mg·cosα

2. Үйкеліс күші: Fүйк = μ·N = μ·mg·cosα

3. Теңдеу:

mg·sinα - μ·mg·cosα = m·a

4. Үдеу:

a = g(sinα - μ·cosα)

5. Сандарды қоямыз:

a = 9.8·(sin30° - 0.1·cos30°) = 9.8·(0.5 - 0.1·0.866) ≈ 4.05 м/с²

Жауабы: 4.05 м/с²

Есеп 2: Импульстің сақталуы

Массасы 0.1 кг доп 20 м/с жылдамдықпен қозғалып, массасы 0.4 кг тыныштықтағы допқа соғылып, бірге қозғалады. Соқтығысудан кейінгі жалпы жылдамдықты табыңыз.

кг

Шешімі:

Импульстің сақталу заңы:

m₁·v₁ + m₂·v₂ = (m₁ + m₂)·v'

Екінші доп тыныштықта болғандықтан (v₂ = 0):

m₁·v₁ = (m₁ + m₂)·v'

Жылдамдықты табамыз:

v' = (m₁·v₁)/(m₁ + m₂)

Сандарды қоямыз:

v' = (0.1·20)/(0.1 + 0.4) = 2/0.5 = 4 м/с

Жауабы: 4 м/с

Үдеу күшке тура, ал массаға кері пропорционал

Күнделікті өмірдегі қолданысы

Тормоз жолы

Автокөліктің тоқтау қашықтығы жылдамдықтың квадратына пропорционал. 2 есе жылдамдық - 4 есе ұзын тормоз жолы.

Поезд қозғалысы

Поезддің қозғалуы мен тоқтауы динамика заңдарына негізделген. Үлкен массаға әсер ету үшін үлкен күш қажет.

Ұшақ қозғалысы

Ұшақтың көтерілуі қанаттарға әсер ететін аэродинамикалық күштердің теңдігіне байланысты.

Жүгіру

Спортшының жылдамдығын арттыру үшін оның қадамының ұзындығы мен жиілігін арттыру қажет.

Жүзу

Суға әсер ету күші жүзушінің алға қарай жылжуына әкеледі (Ньютонның үшінші заңы).

Гольф

Доптың ұшу траекториясы соққы күшіне, соққы бұрышына және ауа кедергісіне байланысты.
